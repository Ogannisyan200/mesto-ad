Финальная проектная работа курса. Приложение Mesto подключено к API-серверу.

🌐 **[Открыть на GitHub Pages](https://ogannisyan200.github.io/mesto-ad/)**


## 🚀 Команды

| Команда | Описание |
|---|---|
| `npm install` | Установить зависимости |
| `npm run dev` | Запустить локальный сервер разработки |
| `npm run build` | Собрать проект (папка `dist`) |
| `npm run deploy` | Опубликовать проект на GitHub Pages |

## ✅ Реализованный функционал

- Получение данных пользователя и карточек с сервера при загрузке
- Редактирование профиля (имя, описание) с отправкой на сервер
- Обновление аватара пользователя
- Добавление новых карточек через API
- Удаление своих карточек (с модальным окном подтверждения)
- Лайки карточек с отображением счётчика
- Просмотр полноразмерного изображения
- Валидация всех форм
- UX-улучшения: индикация загрузки («Сохранение…», «Создание…», «Удаление…»)
- **Вариант 2:** Статистика пользователей по клику на логотип

## 🛠 Технологии

- Vanilla JavaScript (ES6 модули)
- Vite
- CSS (без фреймворков)
- API: [mesto.nomoreparties.co](https://mesto.nomoreparties.co)
- GitHub Actions для автодеплоя
